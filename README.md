# 8086 Assembler Emulation Engine

An automated emulation engine for 8086 assembler projects

### Prerequisites

Docker <br/> 
Node

### Installing

Run inside of the root directory:

```
npm install
```

Run this bash run script to get going:

```
./run.sh
```
